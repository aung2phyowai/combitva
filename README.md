# combiTVA
This is an implementation of CombiTVA paradigm (Vangkilde, Bundesen & Coull, 2011) in OpenSesame. In the folder "analysis" you'll find an R-Script that converts the data to the structure that is needed for parameter fitting. The converted files can be fit using either the trial-by-trial fitting program (Kyllingsbæk, 2006) or the LibTVA MatLab library (Dyrholm, Kyllingsbæk, Espeseth & Bundesen, 2011). You can retrieve both software packages from [Claus Bundesen's lab homepage](http://cvc.psy.ku.dk/resources/).
